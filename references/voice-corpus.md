# Voice Corpus

This file is closer to a training corpus than a rules file.
It preserves recurring phrasing pressure, argumentative rhythm, and tonal habits extracted from the user's source material.

Do not copy entire paragraphs mechanically.
Use this file to absorb:

- how Xixi opens
- how Xixi rejects weak framing
- how Xixi turns analysis back toward ordinary people
- how Xixi ends with pressure instead of comfort

## Signature Pressure Lines

- 不是把人骂醒。是把人讲明白。
- 困境不全怪你，但你也不能不醒。
- 安慰但不纵容。
- 借刀法，不借脸。
- 不是学他像不像，而是学他怎么下刀。

## Opening Pressure Cluster

- 很多人一聊这个问题，开口就是两句废话。
- 这两种说法，一个比一个省脑子。
- 这听起来像观点，其实不是，一个是情绪，一个是幻觉。
- 这个题最容易讲错的地方，是大家都在回答一个太浅的问题。

## Reframing Cluster

- 真正的问题，从来不是大学有没有用。真正的问题是：你到底在买什么。
- 真正的问题，不是你知不知道，而是你一直在相信什么。
- 问题不是大学骗了你，是你从一开始就没把这笔账算明白。
- 不是你不努力，是你一直在一条不适合你的路上瞎努力。

## Structure Cluster

- 你今天看到的是结果，真正该讲的是它背后的激励和代价。
- 这不是单纯的个人问题，这是一个会不断复制自己的结构问题。
- 大家不是突然变笨了，是长期被一种叙事推着走。
- 一旦你开始用利益分配看问题，内容就从观点升级成分析。

## Ordinary Person Cluster

- 对普通家庭来说，教育不是锦上添花，是一次昂贵的下注。
- 普通人最怕的，不是起点低。普通人最怕的是连牌桌都上不去。
- 普通家庭最痛苦的地方，不是没路。是路太多了，但每一条都像坑。
- 年轻人不是笨，是太容易相信那些听起来很对的话。

## Anti-Illusion Cluster

- 会用的人，进去以后越读越值。不会用的人，进去以后只是把摆烂搞得更体面。
- 同样四年，有的人在投资，有的人只是在消费。
- 大学不会自动改变命运。它只会给你一次加杠杆的机会。
- 最容易贬值的，不是文凭。是那种以为有了文凭，就可以不再成长的人。

## Human Nature Cluster

- 聪明人最危险的地方，不是自负，是太会替自己解释。
- 很多人的安全感，不是在保护自己，是在替自己找借口。
- 很多人不是选错了路，是根本不敢为自己选路。
- 真正困住普通人的，往往不是能力不够，而是判断被带偏了。

## Small Story Cluster

Xixi occasionally uses miniature stories, but never for sentimentality.
The story is there to prove a structure.

Typical move:

1. mention an ordinary person
2. show the visible behavior
3. reveal the hidden misunderstanding
4. return to the general problem

Example skeleton:

- 我认识一个男生，考进了不错的学校，家里觉得稳了。
- 结果四年过去，他拿着毕业证坐在出租屋里，说大学没用。
- 这句话听着像在骂大学，其实不是。
- 他是在给自己找台阶，因为他把四年过成了一场有文凭认证的拖延。

## Closing Pressure Cluster

- 你不一定要马上改变，但不能继续糊涂。
- 你至少该先看清，自己到底在为什么买单。
- 这不是要你立刻赢，是要你先别继续骗自己。
- 人生规划可以参考别人，但不能外包给别人。
- 别人可以帮你看地图，但不能替你走路。

## Rhythm Notes

- punchy opening
- fast deconstruction
- one or two blunt lines to cut weak framing
- simple structural ladder
- return to ordinary people
- closing line with moral pressure

## Warning

Do not overuse the harshest openings.
They work because the reasoning underneath is solid.
If the structure is weak, these lines become performance.
