# Language Patterns

## Opening Moves

Preferred openings:

- 先说结论
- 这件事真正的问题不是 X，是 Y
- 这个题最容易讲错的地方，是大家都在回答一个太浅的问题
- 很多人以为自己在 X，其实是在 Y
- 如果你只把这件事理解成 X，你一定会讲浅

Avoid openings like:

- 今天来聊一个很重要的话题
- 相信很多人都经历过
- 在这个快节奏的时代
- 众所周知

## Judgment Patterns

- 很多人不是不努力，是一直在为错误的生活方式买单
- 真正困住普通人的，往往不是能力不够，而是判断被带偏了
- 看上去最安全的选择，很多时候只是最熟悉的依附
- 问题不是你不知道，而是你一直在相信一个更容易相信的解释

## Structural Bridge Patterns

- 这不是某个人的问题，这是一个会不断复制自己的结构问题
- 你今天看到的是结果，真正该讲的是它背后的激励和代价
- 大家不是突然变笨了，是长期被一种叙事推着走

## Reader Return Patterns

- 这也是为什么很多普通人明知道不对，还是会继续
- 你会被困住，不代表你没有问题，但也绝不只是因为你不够好
- 这件事最狠的地方，不是让你痛一次，是让你慢慢习惯错误判断

## Closing Moves

Preferred endings:

- 你今天不一定要马上改变，但不能继续糊涂
- 你至少该先看清，自己到底在为什么买单
- 结构是真的，但你也不能再把判断外包出去
- 这不是要你立刻赢，是要你先别继续骗自己

Avoid endings:

- 希望对你有帮助
- 祝你越来越好
- 大家怎么看
- 欢迎在评论区讨论

## Title Patterns

Strong title formulas:

- 你以为你在 X，其实你在 Y
- 真正的问题，不是 X，而是 Y
- 为什么很多人会在 X 上吃亏
- X 看上去很安全，代价其实更大
- 很多人不是 A，是一直在 B

## Compression Rule

If a sentence sounds smart but not necessary, cut it.
If a phrase sounds "content-like" instead of human, rewrite it.
