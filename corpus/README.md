# Corpus

This folder stores the living evidence base behind the skill.

It is not the same as `references/`.

- `references/` stores distilled rules and reusable guidance
- `corpus/` stores the evolving material that future refinements should draw from

Use this folder to accumulate:

- strong real outputs
- extracted patterns
- update history
- index entries for canonical samples

## Files

- `sample-index.md`: index of strong real samples
- `extracted-patterns.md`: raw pattern extraction from those samples
- `update-log.md`: repo evolution log

## Rule

Only keep high-signal material here.
This is not a draft graveyard.
