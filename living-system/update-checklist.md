# Update Checklist

Use this checklist whenever feeding a new piece back into the repository.

## Quality Gate

- Is this piece clearly stronger than a generic assistant output?
- Does it sound recognizably like Xixi?
- Is there real structure, not just emotion or style?
- Is the ending strong enough to preserve?
- Is at least one line worth reusing as a pattern?

If fewer than 4 answers are yes, do not ingest it yet.

## Extraction Checklist

- Did I extract the best opening line?
- Did I extract the rewritten question?
- Did I extract any new title pattern?
- Did I extract the return-to-reader move?
- Did I extract the closing pressure line?
- Did I note whether Lao Zhou's knife improved the structure?
- Did I note whether XiaoLin-style material improved the explanation?

## Placement Checklist

- Should this become a case in `case-library.md`?
- Should this become a pair in `before-after-pack.md`?
- Should this add lines to `sentence-bank.md`?
- Should this enrich `voice-corpus.md`?
- Should this adjust a format in `platform-playbooks.md`?
- Should this refine `laozhou-engine.md`?
- Should this refine `xiaolin-materials.md`?

## Final Check

- Did I avoid dumping too much raw text?
- Did I preserve the new pattern instead of just archiving a whole draft?
- Did I log the change in `corpus/update-log.md`?
